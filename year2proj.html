My Year Two Project

Description

My second year focused on an MQTT base service with a website front end that allows users to post messages to a music forum updating other users on current events, upcoming concerts & much more. On this forum you can post a message, filter messages into your preferred Genre all with live time updates. The website contained a firestore javascript file which connected to our firebase DB with our website and also provided a delete function within the website, a message javascript file that displayed messages in a table, deleted a message with the id of the same row & Filtered messages based on Genre Finally there was a products javascript file which gets all products from all categories in our DB.

My role in the Project

My role in this project was to work on the Publish and Subscription Javasscript so that when a user wants to post or needs to read the information they have the capability to. This was a new technology for me at the time and there were plenty of bumps in the road as far as getting the code to work went but I got it to work in the end and the wesbite had its main functionality up and running.
